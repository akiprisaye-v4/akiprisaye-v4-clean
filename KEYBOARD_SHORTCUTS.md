# Raccourcis Clavier - A KI PRI SA YÉ

## Navigation Principale

| Raccourci | Action |
|-----------|--------|
| `Tab` | Naviguer vers l'élément suivant |
| `Shift + Tab` | Naviguer vers l'élément précédent |
| `Enter` ou `Espace` | Activer l'élément sélectionné |
| `Échap` | Fermer les modales/panneaux ouverts |

## Liens d'évitement (Skip Links)

| Raccourci | Action |
|-----------|--------|
| `Tab` (au chargement) | Affiche le lien "Aller au contenu principal" |
| `Enter` | Saute directement au contenu principal |

## Accessibilité

| Raccourci | Action |
|-----------|--------|
| `Alt + A` | Ouvrir le panneau d'accessibilité (futur) |
| `Ctrl + +` / `Ctrl + -` | Augmenter/diminuer la taille du texte |
| `Ctrl + 0` | Réinitialiser la taille du texte |

## Navigation dans les formulaires

| Raccourci | Action |
|-----------|--------|
| `Tab` | Passer au champ suivant |
| `Shift + Tab` | Passer au champ précédent |
| `Espace` | Cocher/décocher une case |
| `Flèche haut/bas` | Naviguer dans une liste déroulante |
| `Enter` | Soumettre le formulaire |

## Lecteurs d'écran

### NVDA (Windows)
- `NVDA + Flèche bas` : Mode lecture
- `NVDA + Tab` : Lecture continue
- `H` : Naviguer entre les titres
- `K` : Naviguer entre les liens

### VoiceOver (macOS/iOS)
- `Cmd + F5` : Activer/désactiver VoiceOver
- `VO + Flèches` : Naviguer
- `VO + U` : Ouvrir le rotor (navigation rapide)
- `VO + H` : Naviguer entre les titres

### JAWS (Windows)
- `Insert + F7` : Liste des liens
- `Insert + F5` : Liste des formulaires
- `Insert + F6` : Liste des titres
- `H` : Prochain titre

## Navigation sur la carte

| Raccourci | Action |
|-----------|--------|
| `Tab` | Naviguer entre les markers |
| `Enter` | Ouvrir le popup du marker sélectionné |
| `Échap` | Fermer le popup |
| `+` / `-` | Zoom in/out sur la carte |

## Ti-Panier

| Raccourci | Action |
|-----------|--------|
| `Tab` | Accéder au bouton Ti-Panier |
| `Enter` | Ouvrir/fermer le tiroir Ti-Panier |
| `Suppr` | Supprimer l'élément sélectionné (dans le panier) |

## Conseils pour une navigation optimale

### Pour les utilisateurs de clavier uniquement
1. **Toujours visible** : Le focus est toujours visible avec un contour bleu
2. **Ordre logique** : La tabulation suit l'ordre visuel de la page
3. **Pas de piège** : Vous pouvez toujours sortir d'un élément avec Tab ou Échap

### Pour les utilisateurs de lecteur d'écran
1. **Structure sémantique** : Navigation par titres (H1, H2, H3...)
2. **Labels descriptifs** : Tous les boutons et champs ont des labels clairs
3. **État annoncé** : Les états (ouvert/fermé, actif/inactif) sont annoncés
4. **Zones live** : Les changements dynamiques sont annoncés automatiquement

### Pour les utilisateurs à mobilité réduite
1. **Grandes zones de clic** : Minimum 44x44 pixels pour tous les boutons
2. **Pas de timing strict** : Aucune action n'exige une réponse rapide
3. **Alternatives** : Toutes les actions sont possibles au clavier ET à la souris

## Paramètres d'accessibilité disponibles

Cliquez sur le bouton ♿ en bas à gauche pour accéder au panneau d'accessibilité et configurer :
- Taille du texte (80% à 200%)
- Mode contraste élevé
- Réduction des animations
- Mode daltonien (Protanopie, Deutéranopie, Tritanopie)
